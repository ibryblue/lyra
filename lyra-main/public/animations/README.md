# VRMA 动画文件目录

此目录用于存放VRMA动画文件，这些动画可以应用到VRM角色上。

## 支持的文件格式
- .vrma (VRM Animation格式)

## 使用说明
1. 将VRMA动画文件放置在此目录中
2. 通过网站的VRM控制面板上传或选择动画
3. 选择动画后可以应用到当前的VRM角色上

## VRMA文件获取
您可以从以下地方获取VRMA动画：
- VRoid Studio导出的动画
- VMC协议录制的动作
- 社区分享的动画资源

## 内置动画
系统提供以下内置动画：
- 待机动画 (idle)
- 挥手动画 (wave)
- 点头动画 (nod)
- 跳舞动画 (dance)

## 注意事项
- VRMA文件需要与VRM模型兼容
- 建议动画时长不超过30秒
- 文件大小建议控制在10MB以内
